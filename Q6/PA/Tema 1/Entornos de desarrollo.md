## 1.Qué es un entorno de desarrollo?
Un entorno de desarrollo es un conjunto de herramientas y programas integrados que facilitan la escritura, compilación, depuración y pruebas del software.
$\space$
### 1.1.Entornos para backend y aplicaciones web del lado del servidor
Se suelen usar:
+ Jakarta EE y Spring
+ .NET
+ Ruby on Rails
+ Frameworks de PHP
+ Frameworks de Node
$\space$
### 1.2.Entornos para frontend en aplicaciones web SPA
Se suelen usar:
+ Angular
+ React
+ Vue.js
$\space$
### 1.3.Entorno para aplicaciones nativas desktop y mobile
Se suelen usar:
+ SDK nativos
+ Soluciones cross-plataform, como Flutter o React
$\space$
## 2.Entornos usados en la asignatura
Para el backend usamos varios frameworks de Spring. El acceso a bases de datos relaciones lo hacemos mediante Spring y la Java Persistence API. Para el frontend usamos varios frameworks de React.
